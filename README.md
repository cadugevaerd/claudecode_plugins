# claudecode_plugins

Respositorio de plugins do Cloude Code


## Plugins

| Plugin | O que faz |
| --- | --- |
| `code-debug` | Debug por causa raiz: recebe comando/log, reproduz, coleta evidencias, instrumenta quando necessario e entrega relatorio com causa comprovada e sugestao de fix. |

## Code Debug

Comando Claude Code:

```bash
/code-debug <comando de reproducao ou contexto do erro>
```
