# /update-claude-md - Project Integration

Adds a concise section to your project's CLAUDE.md file with guidance for using Agentc AI Developer Microprocessos in your ongoing work. Integration takes <1 minute.

## What It Does

When you run `/update-claude-md`, the command:

1. **Reads your project context**
   - Looks for README.md generated by `/brief` command
   - Extracts Brief Minimo specification (project name, what agent does)
   - Validates that Microprocesso 1.1 has been completed

2. **Creates concise CLAUDE.md section**
   - Adds â‰¤40 lines of focused guidance
   - Lists available Agentc commands (`/brief`, `/setup-local-observability`)
   - Explains when to use each command
   - Links to plugin documentation for details
   - Adds next steps (Microprocesso 1.3 when available)

3. **Uses progressive disclosure**
   - Keeps CLAUDE.md lightweight
   - Details reference `plugins/agentc-ai-developer/README.md`
   - Skills and commands auto-discovered when plugin is installed

## Using the Command

```bash
/update-claude-md
```

**Prerequisites**:
- You have run `/brief` in this project
- README.md exists with Brief Minimo specification
- Project's CLAUDE.md exists (or will be created)

**Result**:
- New section added to CLAUDE.md (~35 lines)
- Guidance visible to Claude when working on your project
- Commands and next steps clearly documented

## Troubleshooting

**"README.md not found"**
- Run `/brief` first to create your project specification

**"CLAUDE.md doesn't exist"**
- Command creates one automatically with just the Agentc section

**"Section already exists"**
- Command updates existing section instead of duplicating
- Preserves your other CLAUDE.md content
